About GitHub

GitHub, Inc. is a subsidiary of Microsoft which provides hosting for software development and version control using Git. It offers the distributed version control and source code management functionality of Git, plus its own features.

Is GitHub for free!!!

We're happy to announce we're making private repositories with unlimited collaborators available to all GitHub accounts. All of the core GitHub features are now free for everyone. But every developer on earth should have access to GitHub.
